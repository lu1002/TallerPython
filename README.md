Clonar un proyecto usando PyCharm 
1. Abrir PyCharm
2.En la pantalla inicial, haz clic en: Get from Version Control
3.En la ventana que aparece:
En URL, pega la dirección del repositorio que quieres clonar
4. PyCharm automáticamente elegirá la carpeta donde lo guardará
5. Haz clic en Clone
